论文阅读-Delving Deep into Many-to-many Attention for Few-shot Video Object Segmentation

代码:https://github.com/scutpaul/DANet

Abstract

本文解决了 Few-Shot Video Object Segmentation (FSVOS) 的任务，即将查询视频中的对象分割为在几个标记的支持图像中指定的特定类别。关键是对查询视频和支持图像之间的关系进行建模，以传播对象信息。这是一个多对多的问题，通常依赖于计算密集型的full-rank注意力。在本文中，我们提出了一种新颖的Domain Agent Network (DAN)，将full-rank注意力分解为两个较小的注意力。我们将查询视频的单帧视为域代理，在支持图像和查询视频之间架起桥梁。我们的 DAN 允许线性空间和时间复杂度，而不是原始的二次形式，而不会损失性能。此外，我们引入了一种将元学习与在线学习相结合的学习策略，以进一步提高分割精度。我们在 Youtube-VIS 数据集上构建 FSVOS 基准并进行实验，以证明我们的方法在计算成本和准确性方面都优于baseline，实现了最先进的性能。

Introduction

VOS分类

（1）unsupervised VOS：segments primary objects in the videos without any human intervention

（2）semi-supervised VOS：gives the ground truth segmentation of the first frame and propagates the labeled object information into subsequent frames.

FSVOS定义

FSVOS aims to segment new object classes across query videos with only a few annotated support images

FSVOS 旨在仅使用少量带注释的支持图像在查询视频中分割新的对象类

存在的问题

Constructing correlation among support images and query videos is a many-to-many problem and directly using these full-rank attentions in FSVOS suffers from expensive computational cost

过去的解决办法

  • The prototypebased methods，缺点：loses structural information of the support images.
  • The attentionbased methods，缺点：the computational complexity grows exponentially as the number of input images grows.

推断

We hypothesize that the original attention matrix can be replaced with a product of two smaller attention matrices through an agent t.

论文的解决办法

  • we propose a novel Domain Agent Network (DAN),which treats a single frame of the query video as the agent for this video domain.
  • we propose a learning strategy for FSVOS by combining meta-learning with online learning

贡献

  • We delve into the conventional many-to-many attention and prove that the original attention matrix can be replaced by the product of two smaller attention matrices bridged by an agent.
  • We propose a novel domain agent network based on theoretical support, balancing the accuracy, computational burden, and speed
  • We present a learning strategy that combines metalearning and online learning to improve the generalization ability of segmentation and category-specific feature representations.
